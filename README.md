# cookie's cli

## Introduction

一个脚手架，目的是为了解决重复性操作.

## Features

- 创建 react + ts + webpack 模板应用
- 创建 react + qiankun 基座模板
- 创建 react + qiankun 子应用模板
- 初始化 prettier、eslint、stylelint 配置

## Get Started

```javascript
// Global installation 

// yarn
yarn global add cookie-cli 

// npm
npm install -g cookie-cli
```

## Usage

```javascript
cookie create yourAppName // 创建应用

cookie plugin // 初始化配置
```
