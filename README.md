# cookie's cli

## Introduction

一个脚手架，目的是为了解决重复性操作，例如：创建一个 __react__ 项目或者配置 __eslint__ 等等。

## Get Started

```javascript
// Global installation 

// yarn
yarn global add cookie-cli 

// npm
npm install -g cookie-cli
```

## Features

- react-webpack-template
- eslint

## Usage

```javascript
cookie init
```
